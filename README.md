# Financial Risk Analyzer

## 📌 Overview
Financial Risk Analyzer is a machine learning based web application that predicts financial risk using user financial details such as income, loan amount, and credit score.

The project uses a Random Forest Classifier model to analyze financial patterns and classify users into LOW RISK or HIGH RISK categories.

---

## 🚀 Technologies Used

### Frontend
- React.js

### Backend
- Flask

### Database
- MySQL

### Machine Learning
- Python
- Scikit-learn
- Pandas
- NumPy

---

## 🧠 Machine Learning Model
- Random Forest Classifier

The model is trained using financial data and predicts risk based on:
- Income
- Loan Amount
- Credit Score

---

## ✨ Features
- User-friendly dashboard
- Financial risk prediction
- Probability score display
- Interactive frontend
- Real-time prediction

---

## 🔄 Project Workflow

1. User enters financial details
2. Frontend sends data to Flask backend
3. Backend processes the data
4. Random Forest model predicts risk
5. Result is displayed on dashboard

---

## 📊 Prediction Output
- LOW RISK
- HIGH RISK
- Probability Score
